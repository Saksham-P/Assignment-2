About HTML Run

  • This game is designed for desktop but is mobile-ready.
    The minimum play-able witdth for the game is around 700px (landscape),
    below that it becomes hard for 2 players to use the same device.
  • HTML Run is almost like a 2D endless Runner, where each
    player must keep going until they are last one standing
  • The Objective of this game is for each player to survive
    for as long as possible and dodging all obstacles
    until 1 player remains
  • There are 2 power ups players can get within the game
    A sheild which protects player from getting hit by 1 obstacle and
    Spring Boots, which makes the player jump high once.
  • Each player will have 2 controls, and each player will be able
    to choose which button to use for each action.
    Players will have a jump button, and a power button.
    Jump button makes the player jump, and power button uses
    the powerup the player has.
  • To add players (desktop version only), players will need to
    click "Add Player" button on the main page, which will prompt
    the user to set buttons for jump and power for that player.
    Maximum of 5 players can be added.
