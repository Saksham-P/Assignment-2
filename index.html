About HTML Run

  • This game is designed to be mainly played on a desktop with a decent screen size.
    The minimum play-able witdth for the game is around 700px,
    below that it becomes hard for 2 players to use the same device.
  • HTML Run is almost like a 2D endless Runner,
    but after a while the game ends and all surviving players win.
  • The Objective of this game is for each player to survive
    for as long as possible running away from HTML
    until 1 player remains or surviving players reach the end
